# LeadGenMini — Bootstrap Revenue Engine

> Built in a day. Hustling from R500 to revenue. This is MY game.

## What This Is

A lead generation service that scrapes Google Maps + enriches contact data for niche businesses. Sold as $39 custom lists or $99/mo agency subscriptions.

## How It Works

1. **Input:** You specify niche + city (e.g., "plumbers in Austin")
2. **Scrape:** Python/BeautifulSoup extracts business data from Google Maps via DuckDuck
3. **Enrich:** Web scraping extracts emails, phones, websites where available
4. **Deliver:** CSV + VCF files delivered in 24 hours
