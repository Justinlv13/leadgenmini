Fill in:
- **Repository name:** `leadgenmini`
- **Description:** `LeadGenMini — Targeted lead lists in minutes`
- **Public** ✅ (not private)
- **Initialize with a README** ❌ (untick this — we have our own)

Click **Create repository**.

## Step 2: Tell me the repo URL

After creating it, you'll see the repo URL like:
```
https://github.com/YOUR_USERNAME/leadgenmini.git
```
**Copy this URL and share it with me.** Then run the push command below.

## Step 3: Push the code (I'll generate this with your username)

I'll create the push command once you share the repo URL.
OR you can do it manually:

1. Open terminal in this directory
2. Run:
```bash
git remote add origin https://github.com/YOUR_USERNAME/leadgenmini.git
git branch -M main
git push -u origin main
```

## Step 4: Enable GitHub Pages (30 seconds)

1. Go to: `https://github.com/YOUR_USERNAME/leadgenmini/settings/pages`
2. Under "Branch" select `main`
3. Under "Folder" select `/ (root)`
4. Click **Save**
5. Your site will be at: `https://YOUR_USERNAME.github.io/leadgenmini`
